ComSeq

Overview

ComSeq is a set of utilities used for simulating and analyzing compressed-sensing pooling design experiments for next-generation sequencing projects whose purpose is to detect rare allele carriers in a population. The package enables an in-silico simulation of a sequencing pooling experiments, including modeling of sequencing error parameters. The package also enables reconstructing genotypes of individuals profiled in the pools. Detailed instructions and usage examples are given in the README file

About

ComSeq is a collaboration between the Broad Institute, the Open University of Israel , and the Weizmann Institute of Science. If you use the package please cite the following paper:
Rare-Allele Detection Using Compressed Se(que)nsing, N. Shental, A. Amir and O. Zuk (2010)

ComSeq Installation

ComSeq is composed from a set of matlab functions which can be individually used. You can download the whole package, or individual functions below. To install the package simply download the file cs_snps.tgz to a directory of your choice, extract it, and add to your Matlab path. The package uses the GPSR algorithnm, which is available through here . In addition, a pre-compiled script for linux/windwos is supplied below.

applyGPSR.m  -  Runs the GPSR algorithm - using the default parameters.
displaySimulationParameters.m  -  Prints the parameters used for simulations
modifySolution.m  -  Discretize solution: find rounded solution with minimal squared error with respect to measurements
normalizeRows.m  -  Normalize matrix rows
num_lanes_to_max_N.m  -  Compute the maximal sample size we can reconstruct for a given number of lanes
pooling_example.m  -  Simulate a single case of carriers reconstruction
simulateCSseq.m  -  Simulate and reconstruct a genotype vector in a specific setting.
